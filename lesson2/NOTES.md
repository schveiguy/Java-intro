# Instructor notes for Lesson 2 Statements, Declarations, and Variables

## Intro

1. Liken declarations to name assignment and statements to commands (i.e. command
   that 'Fred' stand up, when no 'Fred' is in the room, then "declare" Fred to
   be someone, and tell them to stand up).
2. Explain variables and types with anecdotes. An example is having a box with
   something in it. Maybe a box of coins. Use physical items for more clarity.

## Declaring a variable

Examing the declaration syntax, have the students build code with the `age`
example.

## Print the variable

Add printing to the code.

## Assignment and Math

Show assignment and math. Have the students build some interesting math
equation, maybe quadratic formula? Might be too advanced depending on age.

division.
